## 🎯 เป้าหมาย

- เพิ่มความสามารถให้โปรแกรมอ่านไฟล์ `.csv`
- ห้ามแก้ไข `main_window.py`
- ห้ามแก้ไข `FileLogSource`
- ต้องทำตามหลัก Open/Closed Principle (OCP)

---

## 🛠 สิ่งที่ดำเนินการ

### 1. สร้างโฟลเดอร์

- สร้างโฟลเดอร์ `challenge/`

### 2. สร้างคลาส CsvLogSource

- สืบทอดจาก `ILogSource`
- รับผิดชอบอ่านไฟล์ `.csv` เท่านั้น

### 3. สร้าง LogFactory

- ตรวจสอบนามสกุลไฟล์
- เลือกชนิดของ LogSource ให้เหมาะสม (`.log` / `.csv`)
- คืนค่า instance ที่ถูกต้อง

### 4. แก้ไข main.py

- เรียกใช้งานผ่าน Factory
- ไม่สร้างคลาสโดยตรง

---

## ✅ ผลลัพธ์

- รองรับไฟล์ `.log` และ `.csv`
- ไม่ต้องแก้ไขโค้ดเดิม
- หากต้องการเพิ่มชนิดไฟล์ใหม่ สามารถสร้างคลาสใหม่ได้ทันที
- เป็นไปตามหลัก Open/Closed Principle (OCP)
