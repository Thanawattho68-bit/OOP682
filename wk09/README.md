# นายเขษมศักดิ์ แก่นทน 68114540090

## สัปดาห์ที่ 9 : Object-Oriented Programming (OOP)

### เนื้อหาที่ได้เรียนรู้

- หลักการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming)
- การสร้าง Class และ Object
- การกำหนด Attribute และ Method ภายใน Class
- การใช้ Constructor (`__init__`)
- แนวคิดการสืบทอดคลาส (Inheritance)
- การแยกไฟล์โปรแกรมด้วยโฟลเดอร์ `models`

### สิ่งที่ได้ฝึกปฏิบัติ

- สร้างคลาส `Person`, `Student`, `Staff`
- ทดลองใช้งานการสืบทอดคลาส
- เขียนโปรแกรมแยกไฟล์เพื่อให้ง่ายต่อการจัดการ
- ทดสอบการเรียกใช้งานคลาสจากไฟล์หลัก

### สรุป

การเรียนในสัปดาห์นี้ช่วยให้เข้าใจโครงสร้างของโปรแกรมเชิงวัตถุมากขึ้น  
ทำให้สามารถเขียนโค้ดได้เป็นระเบียบ อ่านง่าย และนำไปต่อยอดกับโปรเจกต์ขนาดใหญ่ได้
